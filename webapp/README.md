# yzzx-frontend Vue 3 + TypeScript + Azure Functions

## Project Structure

- /webapp/src: Vue SPA source code
- /webapp/api: Azure Functions (TypeScript)

## Scripts

- npm run dev: Start local dev server
- npm run build: Build for production
- npm run lint: Lint code
- npm run test: Run unit tests

## Local Development

1. cd webapp
2. npm install
3. npm run dev

## Deploy

- Use Azure Static Web Apps for hosting (CI/CD ready)
- See /scripts and deploy.sh for automation
